# AI-LOAN-PREDICTION-SYSTEM
AI Loan Prediction Model
The AI Loan Prediction Model is a Machine Learning project that predicts whether a loan application will be approved or rejected based on applicant details such as income, education, employment status, loan amount, credit history, and other financial factors.

The project includes data cleaning, preprocessing, exploratory data analysis (EDA), feature engineering, model training, and model evaluation.
It is developed using Python, Pandas, NumPy, Matplotlib, Seaborn, and Scikit-learn in Jupyter Notebook.

Key Features
Data Cleaning & Preprocessing
Exploratory Data Analysis (EDA)
Feature Engineering
Categorical Data Encoding
Machine Learning Model Training
Loan Approval Prediction
Model Evaluation (Accuracy, Precision, Recall, F1-Score, Confusion Matrix)

This project demonstrates a complete end-to-end machine learning pipeline for loan approval prediction. It helps financial institutions automate the loan screening process, reduce manual effort, improve decision-making, and ensure faster, data-driven loan approval.
